\chapter{Tensors and Lagrangians}
We can write the Faraday tensor as,
\begin{equation}
\frac{\partial J^{\mu}}{\partial x^{\mu}} = 0
\end{equation}

\begin{equation}
\frac{\partial F^{\mu \nu}}{\partial x^{\nu}} = \mu_{0}J^{\mu}
\end{equation}

\begin{equation}
\frac{\partial G^{\mu \nu}}{\partial x^{\nu}} = 0
\end{equation}
